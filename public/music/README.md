# Music Files for Recap Page

Please download and place the following music files in this folder:

## Required Files:

1. **chill-background.mp3**
   - For: Intro, Transition, and Outro phases
   - Suggested: Any chill/ambient instrumental music
   - Duration: 2-3 minutes (will loop)
   - Free sources:
     - https://pixabay.com/music/
     - https://www.bensound.com/
     - https://freemusicarchive.org/

2. **hall-of-fame.mp3**
   - Song: "Hall of Fame" by The Script
   - For: Throughout Year 2025 section
   - You need to obtain this legally through:
     - Purchase from iTunes/Amazon
     - YouTube Audio Library (if available)
     - Licensed music services

3. **on-top-of-the-world.mp3**
   - Song: "On Top of the World" by Imagine Dragons
   - For: Within YEP section
   - You need to obtain this legally through:
     - Purchase from iTunes/Amazon
     - YouTube Audio Library (if available)
     - Licensed music services

## Alternative Free Options:

If you prefer to use free/royalty-free music, here are some suggestions:

### For "Throughout Year" section (motivational/inspirational):

### For "Within YEP" section (upbeat/celebration):

### For Background (chill/ambient):
- "Relaxing" by Bensound

## How to Add Files:

1. Download the MP3 files
2. Rename them to match the names above:
   - `chill-background.mp3`
   - `hall-of-fame.mp3`
   - `on-top-of-the-world.mp3`
3. Place them in this folder (`public/music/`)
4. Restart your dev server if needed

## Notes:

- Make sure files are in MP3 format
- Keep file sizes reasonable (under 10MB each if possible)
- Test audio levels are consistent across all tracks
- Ensure you have proper licenses for any copyrighted music
